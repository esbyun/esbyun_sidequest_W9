Project Title: Side Quest Week 9
Ellis Byun, esbyun, 21079246
Description: 
This is a 2D pixel-art platform game prototype. You play as a small fox who can run, jump, and attack across a forest environment. The game 
features tile-based level design, sprite sheet animations, physics-based movement, and background music with sound effects. 

Setup and Interaction Instructions:
Controls:
A/D or left/right arrow keys = move left and right
W or up arrow key = Jump
Space bar = attack

Debug Mode:
/ = toggle debug panel open/closed
G (debug panel must be open) = toggle moon gravity on/off
F (debug panel must be open) = toggle fly / no-clip mode on/off

The debug panel also displays the player's current position, velocity, and gravity value in real time.
